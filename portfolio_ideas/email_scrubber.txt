modules:

- smtplib
- iamp
- selenium
- sqlite

actions(functions)
- add email address to spam list
-   if email address exists in dictionary, remove email 
- remove email from spam list
- add an unsubscribe button
- OOP Classes for functions/methods 

Database for spam addresses:
- SQLite
- json file
